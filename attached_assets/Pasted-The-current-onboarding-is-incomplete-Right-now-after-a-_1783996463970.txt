The current onboarding is incomplete.
Right now, after a user creates an account and finishes setting up their profile, they are immediately dropped into the app without understanding what anything does.
This creates confusion and a poor first impression.
I want a beautiful, interactive, first-time onboarding walkthrough that introduces the app in less than one minute.
Goal
Help new users immediately understand VYV without overwhelming them.
The experience should feel:
Premium
Apple-quality
Calm
Modern
Interactive
Minimal
Friendly
Intentional
The tour should teach users where everything is while letting them interact with the real interface.
Important
Profile creation is NOT the onboarding.
The onboarding starts after the profile has been completed.
Only then should the walkthrough begin.
How it should work
After profile setup:
Open the real app.
Do NOT show another fake screen or slideshow.
Instead:
slightly dim the interface
highlight one real element at a time
display a small floating tooltip
animate smoothly between steps
allow the user to tap Next
allow Skip at any moment
display progress (example: 2 of 5)
The highlighted UI element should remain fully interactive.

UX Rules
Maximum duration:
30–45 seconds
Maximum:
5 steps
Each explanation should be:
One short paragraph.
The user should always understand what to do next.
The onboarding should feel effortless.
Language
Automatically use the user's selected language.
Never mix English and Spanish.
Every tooltip should be translated properly.
First-Time Logic
Create a persistent user property:
has_completed_app_tour
Default:
false
After the user:
finishes the tour
or
taps Skip
update it to:
true
Never automatically show it again.
Settings
Inside:
Profile
→ Settings
→ Help
Add:
Replay App Tour
so users can experience it again whenever they want.
Edge Cases
Never launch before profile completion.
Never repeat after login.
Never repeat after refresh.
If the user closes the app halfway through, resume from the last unfinished step or restart only once.
Ensure the walkthrough works even if the user hasn't connected Spotify, YouTube, Apple Health, or Google Calendar yet.
VYV Philosophy
Every design decision should reinforce this idea:
Technology should help people spend less time on their phones and more time living real life.
The onboarding should communicate that VYV is not another social network.
It is an intentional life companion.
The experience should leave users thinking:
"I understand this app in under a minute, and I already know why it's different."
